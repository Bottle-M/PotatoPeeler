# PotatoPeeler
A simple tool to remove unused chunks of my Minecraft server, freeing disk space.  

## Outline

1. unsuitable-scenarios
2. 对于原版存档格式可以像这样指定不同维度的数据目录：`world`, `world/DIM-1`。
3. 不会移除被 `/forceload` 强制加载的区块。
